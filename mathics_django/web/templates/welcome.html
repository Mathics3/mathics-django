Mathics is a general-purpose computer algebra system.

Enter queries and evaluate them by pressing Shift + Return. See the gallery for some examples or the documentation for a full list of supported functions. See /about for software version, machine, and system information.

Mathics uses MathJax and three.js to display beautiful math and 3D graphics. For further information about the project visit the home page.
